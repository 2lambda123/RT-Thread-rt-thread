# STM32F103C8 Blue Pill BSP Introduction

## Onboard Resources

- MCU：STM32F103C8T6 @ 72MHz, 64KB FLASH , 20KB RAM
- Peripherals
  - LED：PC13
- Debug IO interface type: SWD
